Project Description

For our project, we had developed a database of string prompts that will be hosted on a remote server which is connected to a webpage that allows users to get an image from the server and the option to download the image afterwards. Within the String List, there are 10 total string prompts that can be viewed through the carousel by clicking on the at the sides or the dots at the bottom. Once the user finds the appropiate string prompt, they can type the prompt in the textbox and pressed enter.


When the button is pressed, there are two results that will occur: it will either fail or the string is sucessfully received and the server will send back an image. There is a custom weather API built on the website to view the weather and temperature ranging from Saturday to Friday. At the very bottom of the page, there are optional settings the user can choose to satisfy their viewing experience. They can either change the background-color, listen to music or, enable sound effects on buttons.


Please note: You must install Node JS in order to connect to the database. Then open the terminal and type "node server.js".
